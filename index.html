Welcome to MyYouTubeLibraty

With this library you can easily create a c# project that uses youtube search list web service. Given a valid youtube videoID, you will get information about suggested playlists from youtube including their VideoID, title and description.

Getting Started

In order to use youtube API you must first register your project at Google Developers Console (console.developers.google.com) with your Google Account, and generate an API key. Next step is to download this library and add a reference to it in your project. At the top of your page include : using MyYouTubeLibrary;

Creating objects

You will need to use few classes: IyouTubeDataService YouTubeService(String DataType,String videoID ,String APIkey)

  • DataType - the only valid argument at the moment is "VideoList", that creates a proper video list search method. In the future it is possible that we add more services to this library.
  • videoID - a viseoID you would like to search. this could be found in the query string of any youtube video URL. example: https://www.youtube.com/watch?v=06_ng5caZ78
  • APIkey - you API key from Google Developer Console.

Code example:

YouTubeDataServiceFactory factory = new YouTubeDataServiceFactory(); IyouTubeDataService func1 = factory.YouTubeService("VideoList", "bdOTUocn5w", "AIzaSyAs5EBJ96dISdxrgU1y5v93lYqIx-9Zbs0"); YouTubeData data = func1.getYouTubeData("VideoList", "_bdOTUocn5w", "AIzaSyBGKoPpk2OE3Z-cfCwF035g7ljgZe25wo");

The returned object contain a c# List and you can apply any methods on it to use the information however you like. Each element of the list is an object from class Item, it contain: string videoId { get; set; } string title { get; set; } string description { get; set; }
